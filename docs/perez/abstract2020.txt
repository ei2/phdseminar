En esta charla se estudia un método de resolución heurístico adaptivo para el problema de ruteo de vehículos con puntos de transbordo y ventanas de tiempo. El problema fue formulado para una empresa consultora de logística danesa. En este caso en particular se considera un problema en que cada nodo proveedor está asociado a su correspondiente nodo cliente y se considera un único punto de transbordo para todos los nodos. Se propone un algoritmo heurístico de resolución que permite encontrar soluciones a problemas de gran tamaño en tiempos acotados. El algoritmo implementa un acercamiento basado en control de parámetros que permite adaptar las tasas de uso de operadores permitiendo encontrar resultados de muy buena calidad.
